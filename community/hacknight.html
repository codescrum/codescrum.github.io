Join us for our regular Ruby Hacknight on the 3rd Tuesday of each month. We'll start around 18:30 with pizza and drinks, and then we can kick off the hacking!

About the Hacknight

The hacknight is all about discussing interesting Ruby topics, as well as working on (pet) projects.

Format

The format is in general very open, but like many things it works best with some organisation ahead of time. If you have a topic you’d like to discuss, a project that interests you or a gem you like to work on, please bring a 3 minute introduction.

The Event

An open, relaxed meetup for all levels of web developers to come and talk shop, share knowledge, collaborate and hack on all manner of web and mobile projects.

Join us

The only requirements are a healthy interest in web or mobile programming and the desire to dive right into a hack. Join us for our regular Ruby Hacknight on the 3rd Tuesday of each month. We'll start around 18:30 with pizza and drinks, and then we can kick off the hacking!
